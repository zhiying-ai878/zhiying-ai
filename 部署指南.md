## 2. 部署到 Vercel

### 步骤 1: 登录 Vercel
- 访问 [Vercel](https://vercel.com/) 并登录
- 点击 "Add New" -> "Project"

### 步骤 2: 导入项目
- 选择 "Import Git Repository"
- 输入 GitHub 仓库地址: `https://github.com/zhiying-ai878/zhiying-ai.git`
- 点击 "Import"

### 步骤 3: 配置部署
- **Project Name**: 保持默认或自定义
- **Framework Preset**: 选择 "React"
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Environment Variables**: 无需额外配置

### 步骤 4: 部署
- 点击 "Deploy"
- 等待部署完成
- 部署成功后，会获得一个 Vercel 域名

## 3. 部署到 Netlify

### 步骤 1: 登录 Netlify
- 访问 [Netlify](https://www.netlify.com/) 并登录
- 点击 "Add new site" -> "Import an existing project"

### 步骤 2: 导入项目
- 选择 "GitHub"
- 搜索并选择 `zhiying-ai878/zhiying-ai` 仓库
- 点击 "Import"

### 步骤 3: 配置部署
- **Build command**: `npm run build`
- **Publish directory**: `dist`
- 点击 "Deploy site"

### 步骤 4: 部署
- 等待部署完成
- 部署成功后，会获得一个 Netlify 域名

## 5. 故障排除

### 常见问题
1. **API 代理问题**
   - 检查 Vercel/Netlify 的代理配置
   - 确保所有数据源都使用正确的代理路径

2. **数据获取失败**
   - 检查网络连接
   - 查看浏览器控制台的错误信息
   - 验证数据源API是否正常

3. **部署失败**
   - 检查构建日志
   - 确保依赖安装正确
   - 验证 build 命令是否执行成功
