## 🤖 三、如何使用本地AI功能

### 步骤1：打开本地AI页面
点击左侧菜单的 **"本地AI"** 按钮

### 步骤2：选择AI模型类型
在训练配置中选择：
- `simpleNN` - 简单神经网络（快速）
- `deepNN` - 深度神经网络（推荐）
- `lstm` - LSTM网络
- `ensemble` - 集成模型

### 步骤3：配置训练参数
- **Epochs（训练轮数）**: 50-100
- **Batch Size（批次大小）**: 32
- **Learning Rate（学习率）**: 0.001

### 步骤4：开始训练
点击 **"开始训练"** 按钮
- 观看训练进度
- 查看损失曲线
- 等待训练完成

### 步骤5：使用AI预测
训练完成后：
- 输入股票代码
- 询问AI建议
- 查看预测结果
- 获取技术指标分析

## 📊 四、如何集成真实数据源

### 重要说明
⚠️ **东方财富没有公开API！**

### 可用的数据源

#### 方案1：新浪财经API（已集成✅）
- 免费使用
- 实时行情
- 自动使用

#### 方案2：腾讯财经API（已集成✅）
- 免费使用
- 数据全面
- 可切换使用

#### 方案3：模拟数据（已集成✅）
- 完全可控
- 适合开发测试
- 自动降级使用

### 数据源切换（开发者）

在代码中使用：

```typescript
import { getStockDataSource } from './utils/stockData';

// 使用新浪财经（默认）
const dataSource = getStockDataSource('sina');

// 使用腾讯财经
const dataSource = getStockDataSource('tencent');

// 使用模拟数据
const dataSource = getStockDataSource('mock');
```

## 📱 五、如何与东方财富配合使用

### 推荐工作流程

#### 第1步：在智盈AI中分析
1. 打开 http://localhost:3003/
2. 查看市场行情
3. 使用本地AI进行分析
4. 获取AI交易建议

#### 第2步：在东方财富中验证
1. 打开东方财富APP或客户端
2. 搜索相同股票
3. 对比行情数据
4. 研究基本面信息

#### 第3步：手动做出决策
1. 综合AI建议和自己的判断
2. 决定买入/卖出/持有
3. 确定交易数量和价格

#### 第4步：在东方财富中交易
1. 打开东方财富交易界面
2. **手动输入交易指令**
3. 确认交易信息
4. 提交订单

#### 第5步：记录和回测
1. 在智盈AI中记录交易
2. 查看交易历史
3. 分析交易结果
4. 优化AI策略
