# 智盈AI - 全面自动化优化 - 产品需求文档

## Overview
- **Summary**: 对智盈AI投资系统进行全面的自动化优化，实现全市场主力自动追踪、热点自动追踪、行情自动分析、买卖信号自动汇报，无需用户手动开启，让系统24小时自动运行。
- **Purpose**: 解放用户双手，让AI系统自动完成所有市场监控和分析工作，用户只需接收信号和做出决策，大幅提升投资效率。
- **Target Users**: 股票投资者、量化交易爱好者、需要AI辅助决策的投资用户。

## Goals
- 实现全市场主力自动追踪功能（自动监控所有股票的主力资金流向）
- 实现热点自动追踪功能（自动发现和追踪市场热点板块和个股）
- 实现行情自动分析功能（自动分析技术指标、趋势、形态）
- 实现买卖信号自动汇报功能（自动生成并推送买卖信号）
- 所有功能无需用户手动开启，系统登录后自动运行
- 优化所有现有功能的用户体验和性能

## Non-Goals (Out of Scope)
- 不实现真实资金交易（仅模拟交易）
- 不接入真实的实时行情API（继续使用模拟数据）
- 不实现后端服务器部署
- 不包含移动端原生应用开发

## Background & Context
- 现有项目已完成基础功能开发，包含11个核心页面
- 技术栈：React + TypeScript + Vite + Ant Design + ECharts
- 界面已优化：统一使用Card size="small"、bordered={false}、padding: 0
- 当前功能需要用户手动操作，缺乏自动化能力
- 需要创建一个全局状态管理系统来协调所有自动功能

## Functional Requirements
- **FR-1**: 全市场主力自动追踪 - 系统自动监控所有股票的主力资金流向，实时更新
- **FR-2**: 热点自动追踪 - 自动发现市场热点板块和个股，追踪热度变化
- **FR-3**: 行情自动分析 - 自动分析技术指标、趋势线、形态分析
- **FR-4**: 买卖信号自动汇报 - 基于AI分析自动生成买卖信号并推送通知
- **FR-5**: 全局自动化引擎 - 创建统一的自动化调度系统，无需用户手动开启
- **FR-6**: 实时通知系统 - 实现桌面通知、声音提醒等多种通知方式
- **FR-7**: 历史记录系统 - 自动记录所有AI分析结果和信号历史
- **FR-8**: 现有功能优化 - 优化所有现有页面的性能和用户体验

## Non-Functional Requirements
- **NFR-1**: 性能优化 - 自动更新频率可配置，不影响界面响应
- **NFR-2**: 可靠性 - 自动功能出错时不影响系统整体运行
- **NFR-3**: 可配置性 - 用户可配置自动更新频率、通知方式等
- **NFR-4**: 用户体验 - 界面清晰展示所有自动运行的状态
- **NFR-5**: 数据持久化 - 自动保存分析历史和信号记录

## Constraints
- **Technical**: React + TypeScript + Vite + Ant Design + ECharts
- **Business**: 模拟交易系统，学习和研究用途
- **Dependencies**: 使用模拟数据，无外部API依赖

## Assumptions
- 用户保持浏览器页面打开以运行自动功能
- 使用模拟数据进行功能演示
- 用户使用现代浏览器访问
- 系统在前台运行时执行自动任务

## Acceptance Criteria

### AC-1: 全市场主力自动追踪
- **Given**: 用户已登录系统
- **When**: 系统自动运行
- **Then**: 自动监控所有股票的主力资金流向，超大单、大单、中单、小单数据实时更新
- **Verification**: `human-judgment`
- **Notes**: 验证主力追踪页面数据自动更新

### AC-2: 热点自动追踪
- **Given**: 用户已登录系统
- **When**: 系统自动运行
- **Then**: 自动发现市场热点板块和个股，显示热度排名和变化趋势
- **Verification**: `human-judgment`
- **Notes**: 验证首页热点股票自动更新

### AC-3: 行情自动分析
- **Given**: 用户已登录系统
- **When**: 系统自动运行
- **Then**: 自动分析技术指标（MACD、KDJ、RSI等）、趋势线、形态分析
- **Verification**: `human-judgment`
- **Notes**: 验证股票详情页面自动分析

### AC-4: 买卖信号自动汇报
- **Given**: 用户已登录系统
- **When**: AI发现买卖信号
- **Then**: 自动生成信号，显示置信度、理由，并推送通知
- **Verification**: `human-judgment`
- **Notes**: 验证信号提醒页面自动更新

### AC-5: 全局自动化引擎
- **Given**: 用户登录系统
- **When**: 进入主页面
- **Then**: 所有自动功能自动启动，无需用户手动开启
- **Verification**: `human-judgment`
- **Notes**: 验证登录后系统自动运行

### AC-6: 实时通知系统
- **Given**: 系统发现重要信号
- **When**: 信号生成时
- **Then**: 显示桌面通知、声音提醒，右上角铃铛图标显示未读数量
- **Verification**: `human-judgment`
- **Notes**: 验证通知功能正常

### AC-7: 历史记录系统
- **Given**: 系统运行一段时间
- **When**: 用户查看历史记录
- **Then**: 显示所有AI分析结果和信号历史，可追溯
- **Verification**: `human-judgment`
- **Notes**: 验证历史记录功能

### AC-8: 现有功能优化
- **Given**: 用户使用系统
- **When**: 浏览各个页面
- **Then**: 所有页面加载快速，交互流畅，数据展示清晰
- **Verification**: `human-judgment`
- **Notes**: 验证所有现有功能优化
