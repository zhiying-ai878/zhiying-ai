# 智盈AI

基于AI的股票交易软件，实现实时监控主力动向、自动量化交易等功能。

## 功能特性

- 📊 实时股票行情监控
- 🤖 AI主力动向分析
- 🔄 自动量化交易
- 🔐 交易账户安全登录
- ⚙️ 交易参数自定义设置
- 📈 多维度技术指标分析

## 技术栈

### 前端
- React + TypeScript
- Ant Design + ECharts
- Redux + Saga 状态管理

### 后端
- Node.js + Express
- Python (AI分析模块)
- MongoDB 数据存储

### 第三方服务
- 股票行情API
- 交易接口API

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 项目结构

```
src/
├── components/          # 通用组件
├── pages/              # 页面组件
│   ├── Dashboard/      # 仪表盘
│   ├── StockDetail/    # 股票详情
│   ├── Trade/          # 交易页面
│   ├── Settings/       # 设置页面
│   └── Login/          # 登录页面
├── services/           # API服务
├── store/              # 状态管理
├── utils/              # 工具函数
└── AI/                 # AI分析模块
```

## 核心功能

1. **主力监控**：实时跟踪主力资金流向
2. **自动交易**：基于AI策略的自动买卖
3. **风险控制**：设置止损止盈参数
4. **交易记录**：完整的交易历史记录
5. **数据分析**：多维度技术指标分析

## 安全说明

- 交易账户信息加密存储
- 所有API请求使用HTTPS
